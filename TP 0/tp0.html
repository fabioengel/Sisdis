Trabalho Prático 0

O trabalho prático 0 (TP0) consiste na utilização da biblioteca SMPL (Simulation Programming Language). Esta biblioteca permite a simulação de sistemas arbitrários, sendo uma extensão funcional da linguagem de programação C. Em conjunto com a linguagem C, o smpl compõe uma linguagem de simulação orientada a eventos.

O TP0 é composto por 4 tarefas, sendo:

  • Tarefa 0: Editar, compilar e executar "tempo.c".
    • Baixe o conteúdo de Tarefa 0 e execute no terminal: "make -f makefile" e "./tempo <numero de nodos>". 
  • Tarefa 1: Cada nodo sem-falha testa o seguinte no anel.
    • Link para Tarefa 1. Link para log de exemplo.
  • Tarefa 2: Cada nodo sem falha executa testes sequencialmente até encontrar outro modo sem-falha.
    • Link para Tarefa 2. Link para log de exemplo.
  • Tarefa 3: Cada nodo mantém localmente o vetor state e atualiza seus valores ao testar o nodo (só atualizada entrada corresponde ao nodo em que testou).
    • Link para Tarefa 3. Link para log de exemplo.
  • Tarefa 4: Cada nodo sem falha ao testar outro nodo sem falha obtém informações sobre aqueles nodos que não testou.
    • Link para Tarefa 4. Link para log de exemplo.
